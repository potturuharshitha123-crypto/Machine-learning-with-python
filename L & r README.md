# Diabetes Regression using Lasso and Ridge Regression

## 📌 Overview

This project demonstrates the implementation of **Lasso Regression** and **Ridge Regression** using the built-in Diabetes dataset from Scikit-learn. The models are trained and evaluated using **Mean Squared Error (MSE)** to compare their prediction performance.

---

## 📂 Dataset

- **Dataset:** Diabetes Dataset
- **Source:** Scikit-learn (`load_diabetes()`)
- **Number of Samples:** 442
- **Number of Features:** 10
- **Target Variable:** Disease progression measure

---

## 🛠️ Technologies Used

- Python
- Scikit-learn
- NumPy

---

## 📚 Algorithms

- Lasso Regression (L1 Regularization)
- Ridge Regression (L2 Regularization)

---

## 📋 Project Files

- `lasso_regression.py` – Implements Lasso Regression.
- `ridge_regression.py` – Implements Ridge Regression.
- `lasso_vs_ridge.py` – Compares both models using Mean Squared Error.

---

## ▶️ Steps Performed

1. Import required libraries.
2. Load the Diabetes dataset.
3. Split the dataset into training and testing sets.
4. Train the regression model.
5. Predict the target values.
6. Evaluate the model using Mean Squared Error (MSE).
7. Compare the performance of Lasso and Ridge Regression.

---

## 📊 Evaluation Metric

- **Mean Squared Error (MSE)**

A lower MSE indicates better prediction performance.

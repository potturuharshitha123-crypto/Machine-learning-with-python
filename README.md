# Logistic Regression using Scikit-learn

## Overview
This project demonstrates a basic implementation of Logistic Regression using Python and Scikit-learn.

## Topics Covered
- Importing libraries
- Loading dataset using Pandas
- Feature and target selection
- Splitting the dataset using train_test_split()
- Training the Logistic Regression model
- Making predictions
- Evaluating the model using accuracy_score()

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Google Colab

## Learning Outcome
Learned how Logistic Regression is used for binary classification problems such as Pass/Fail, Yes/No, and True/False predictions.
